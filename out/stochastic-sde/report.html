Stochastic Differential Equations + 3 ML algorithms

Euler–Maruyama SDE engine with system identification, ensemble filtering, and score-based diffusion.

What this run covers

Models dX = f(X,t)dt + g(X,t)dW where the solution is a random process. Three machine-learning paradigms run on it: (1) maximum-likelihood SDE parameter recovery (system identification); (2) an Ensemble Kalman Filter that estimates a hidden motor current from noisy speed-only measurements (filtering/inference); (3) a denoising-diffusion generative model that learns a bimodal target and samples it by integrating the reverse-time SDE.
